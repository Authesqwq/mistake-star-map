# 错题星图 — 产品概述

## 产品背景

学生在日常学习中积累了大量错题。传统错题本容易变成静态收藏夹，核心问题在于：

- 复练率低：学生不知道"今天该练哪道题"
- 错因不清：知道错了但不知道为什么错
- 推荐不解释：练习推荐缺少可理解的依据

## 产品目标

把静态错题本转化为可执行的学习闭环：

1. 录入错题 → 2. AI 归因 → 3. 用户校正 → 4. 知识点图鉴点亮
→ 5. 今日三题推荐 → 6. 复练答题 → 7. 反馈与掌握度更新
→ 8. 成就激励 → 9. 学习报告 → 循环

## 核心用户路径

1. 进入今日修复中心，了解当前需要优先处理的知识点
2. 录入错题（手动填写或使用示例题）
3. 运行 AI 归因，识别知识点和错因标签
4. 校正并确认归因结果
5. 查看知识点图鉴，了解各知识点的掌握状态
6. 在今日修复中心开始复练任务
7. 答题提交，查看反馈和掌握度变化
8. 查看成就中心和学习报告

## MVP 范围

- 完整前后端 monorepo
- AI 归因 + fallback
- 规则推荐 + 规则判题 + 规则掌握度 + 规则成就
- 所有业务数据存储在 localStorage
- 不依赖数据库、用户系统、第三方服务

## 非目标

- 不做真实 OCR
- 不做真实数据库持久化
- 不做多学科支持（当前仅数学）
- 不做排行榜、PK、抽奖
- 不做教师/家长端
- 不替代教师判断
